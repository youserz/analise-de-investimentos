# Pai da Análise de Investimentos

Este projeto é um repositório de estudos sobre a vida, obra e legado de
Benjamin Graham, reconhecido como o "Pai da Análise de Investimentos".

## Objetivo
Fornecer uma base sólida para distinguir o comportamento especulativo
do investimento inteligente, pautado na análise lógica e na disciplina emocional.

## Conteúdo
- Conceitos fundamentais de value investing
- Princípios de margem de segurança
- Disciplina emocional no investimento
- Reflexões baseadas nas obras de Benjamin Graham

## Fonte de estudo
Conteúdo originalmente estruturado com apoio do NotebookLM.
